Purpose

A single shared tool that lets the Customer Experience team log every customer complaint, route it to the right Responsible Party, and report up to the GM four times a day (9 AM · 12 PM · 4 PM · 5 PM COB) — replacing scattered WhatsApp messages, Excel sheets and email threads with one synced source of truth.

GM / Leadership
Receives the 9 AM, 12 PM and 4 PM Daily Email Report via Outlook.
3

Core Success Criteria

✓
Every complaint is logged in the app within 5 minutes of being received — no more Excel side-files.
✓
One source of truth — Manager, all 5 Agents and the Daily Email Report all read from the same SharePoint list via Power Automate.
✓
Every ticket has a Responsible Party assigned — zero unassigned tickets at COB.
✓
≥ 80 % resolution rate on tickets received in the same business day (rolling 7-day average).
✓
9 AM, 12 PM, 4 PM and 5 PM reports sent to GM on time — generated and copied to Outlook within 10 minutes of the slot.
✓
Agent slot reports posted to WhatsApp at every Weekday 2 PM / 5 PM and Weekend 1 PM / 4 PM checkpoint.
✓
No data loss — tickets survive browser refresh, device swap, and offline pockets (local cache + cloud sync).
✓
No credentials in source code — Power Automate URLs stored on each device only via localStorage; all data stays within the company's Microsoft 365 environment.
4

Functional & Operational Checks

✓
Managers Hub can log, edit, bulk-update, sort, filter, search every complaint; export JSON, CSV and Excel.
✓
Agent shifts set by Love (Day / Night / Off / Leave) sync to every Agent's device.
✓
Cumulative report rolls up tickets by Responsible Party with % resolution; party list is editable in-app.
✓
Slot Reports produce a one-click WhatsApp message, a styled PDF and a 4-sheet Excel export.
✓
Daily Email Report auto-pulls live cloud data and is copy-pastable into Outlook with full formatting.
✓
Works on Android, iOS and desktop browsers — no app install, no backend, no IT ticket.
✓
Failure is visible — sync errors show a banner; no silent data drops.
✓
Zero extra cost — runs entirely on Microsoft 365 (Power Automate + SharePoint) already licensed by the company; no third-party cloud services.
